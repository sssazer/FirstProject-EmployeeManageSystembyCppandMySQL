# FirstProject-EmployeeManageSystembyCppandMySQL
基于控制台创建的职工管理系统，主要使用语言为C++，数据存放在MySQL数据库中

- 使用方式：

下载这个文件夹之后，直接点击.sln即可打开项目工程（需要VS2019或更高的版本）

- 注意事项：

  - 要先将名为manage_system_db的数据库创建好。（表结构见source_code分支中的SqlCode文件，或者直接使用Database File中的manage_system_db数据库文件）
  
  - 在三个类的构造函数中都有登录MySQL数据库的操作，要记得把账号密码改成自己的
